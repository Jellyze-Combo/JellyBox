# JellyBox

这个项目是和朋友一起研发的，市面上干燥盒子多种多样，但是各有各的缺点，有的不能边烘干边打印，有的只能作为一个存放箱子使用，还有的烘干逻辑模糊，总之，没有一个完整的、一站式的解决方案，正如鲁迅所言，没有的东西，那就给他造出来！于是JellyBox项目孕育而生。

# 理想中的干燥箱

在项目发起之前，我采集了各个群里大佬的意见和想法，并结合自身的使用，提出了这些要求。

1. 闭环温度控制烘干系统

一个干燥箱如果不能干燥，那和箱子有什么区别？一个干燥箱如果不能做到全材料烘干，那和ams2 pro有什么区别？因此设计的第一个理念就是全材料干燥，无论是常规的pla还是petg，还是工程塑料abs和pc，都可以进行干燥。而目前市面上的干燥箱几乎都是开环，设置好温度后就直接开始加热，有的配置了温度显示但是温度并没控制系统，因此这次也是加入了温度控制系统实现闭环的温度控制，并实现温湿度的直接显示。提供多种材料预设，可以很方便的调整材料。

2. 安全的使用体验

很多人都有白天打印晚上烘干的时候，因此安全性至关重要，毕竟谁也不想睡觉的时候屋子着火吧（XD..)，设计上加热组件采用PTC陶瓷加热片，温度稳定安全，不容易着火，取消常见的电热丝加热，并加入温度检测系统，当温度异常的时候可以警报和断电，并在电路设计上加入反接保护等，降低误操作导致意外的可能性。

3. 自动风门

很多干燥箱是没有自动风门的，或者应该说95%的干燥箱都不具备自动风门，但这个功能其实很重要。当加热干燥的时候，耗材中的水份会变成水蒸气，这个时候耗材是干燥的，但是当加热停止后，水汽并不会消失，因此温度降低后，水蒸气会重新凝结被耗材吸收，通常情况下在干燥箱内放置干燥剂可以解决，但是毕竟不够智能化，而且干燥剂也会失效，因此自动风门依然是一个必备的功能。

4. 噪音小

不只是打印机，干燥箱的静音也很重要，因此需要通过一些设计去减少干燥箱运行时的声音，在不添加额外配件并保持简单的情况下，可以通过设计吸音结构来减少噪音，目前测试发现稀疏填充中的螺旋填充有较好的吸音效果，在特定位置采用螺旋填充可以减少部分噪音。

5. 料盘旋转

因为吹风的位置是固定的，会出现靠近热风口的一端干燥程度更高，而远离的地方干燥程度低的情况，而且局部温度不一在有些时候会导致线材的直径偏差大，在高温烘干时还可能会出现局部耗材软化的情况，加入料盘旋转可以在烘干过程中均匀受热，从而增加烘干的均匀性。

6. 供料系统

AMS的易用性得到广泛好评，而1600的高昂价格对很多不那么需要多色打印的人而言确实是有一点贵，那么，为什么不diy一个多色换料系统呢？目前的多色系统如兔子多色等复杂度高，可靠性差，而且联动性一般，jellybox使用esp32和mqtt协议，能够进行无线控制和联动，更好的服务于打印。
